To compile each of the .c files, run the command

    gcc Qx.c -std=c99 -o Qx.exe

Where Qx is the file name
Each question answer is in its own .c file. 
Qx ranges for Q1 to Q6.

To run each of the files, run the command 

    ./Qx.exe
